Installation

Before you start

  • Current documentation was created to help you with quick installation and configuration of VMeet web. Please, read it carefully to avoid most of potential problems with incorrect configuration.

  • Below chapter describes how to install visual studio code & NodeJs

  • Once you will have activated the theme you will need to change some of the pages and settings for optimal performance. That’s why, please, do not contact the Support centre beforehand. Read firstly the documentation, implement all the steps following the instructions and only after that, if the issues persist, contact us.

Change Welcome Page Text:

1 - Edit the “body.html” file (It’s blank on fresh installs)
sudo vi /usr/share/jitsi-meet/body.html
_

2 - Add references to your own CSS and JS.

_

3 - Create custom.css and save it with the new styles
sudo vi /usr/share/jitsi-meet/css/custom.css
In this case you want to change the home page text:
/usr/share/jitsi-meet/css/custom.css
.welcome .header .header-text-title { display:none !important }
.welcome .header .header-text-subtitle:before { content:‘Woooo Meet’; color: #fff; font-size: 2.5rem; font-weight: 500; line-height: 1.18; display: block; }

sudo vi /usr/share/jitsi-meet/custom.js
